Tentang SiBindo

SiBindo Translator adalah purwarupa sistem penerjemah Bahasa Isyarat Indonesia (BISINDO) berbasis webcam yang dikembangkan sebagai Tugas Akhir Program Studi D-III Teknik Informatika, Politeknik Negeri Bengkalis. Tujuannya: menjembatani kesenjangan komunikasi antara penyandang tunarungu dan masyarakat dengar melalui Computer Vision dan Deep Learning.

Arsitektur Sistem

Sistem mengadopsi pendekatan Two-Stage Detection System — setiap frame webcam melewati lima tahap pemrosesan secara berurutan.

1
YOLOv8
Deteksi Region of Interest (bounding box penutur/tangan)
2
MediaPipe
Ekstraksi 21 landmark tangan × 3D × 2 tangan = 126 fitur per frame
3
Normalizer
Translasi wrist-relative + scale invariance (invariant terhadap posisi & jarak kamera)
4
LSTM
Klasifikasi sekuensi 30 frame → label kata dengan skor confidence
5
NLP rule-based
Smoothing (3× konsekutif), dedup, sliding window 30 kata → kalimat
6
gTTS
Konversi teks → suara (Bahasa Indonesia) untuk komunikasi dua arah

Teknologi

Python 3.12 Flask TensorFlow 2.x Ultralytics YOLOv8 MediaPipe Tasks API OpenCV NumPy scikit-learn gTTS Flask-Limiter Docker Gunicorn
